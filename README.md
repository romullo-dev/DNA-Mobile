# 📱🚚 DNATransportes Mobile

Aplicativo mobile desenvolvido para acompanhamento e atualização de operações logísticas em tempo real.

O DNATransportes Mobile é a extensão da plataforma web, permitindo que motoristas e operadores registrem e acompanhem o status das entregas diretamente pelo celular.

---

## 🎯 Problema

Na logística, grande parte das atualizações de entrega ainda depende de comunicação manual, o que gera:

- Falta de atualização em tempo real  
- Ruído na comunicação  
- Baixa rastreabilidade das entregas  
- Dificuldade no acompanhamento operacional  

---

## 💡 Solução

O aplicativo mobile permite que as informações sejam registradas diretamente na operação, garantindo:

- Atualização em tempo real  
- Maior controle das rotas  
- Registro de ocorrências  
- Comunicação direta com o sistema  

---

## ⚙️ Principais funcionalidades

- 📦 Visualização de rotas atribuídas  
- 🚚 Atualização de status das entregas  
- 📍 Registro de localização  
- 📸 Upload de fotos (comprovante de entrega)  
- 📝 Registro de ocorrências  
- 🔄 Sincronização com API backend  

---

## 🛠️ Tecnologias utilizadas

- React Native (Expo)
- JavaScript
- Axios (consumo de API)
- AsyncStorage
- API REST (Laravel)

---

## 🔗 Integração com o sistema

O aplicativo se comunica com o backend do DNATransportes, permitindo:

- Consulta de rotas  
- Atualização de status  
- Envio de dados em tempo real  
- Persistência das informações  

---

## 🧠 Arquitetura

- Comunicação via API REST  
- Gerenciamento de estado no app  
- Armazenamento local com AsyncStorage  
- Estrutura modular para escalabilidade
